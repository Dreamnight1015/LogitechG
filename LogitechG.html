프로젝트명 - Logitech G 홈페이지 리뉴얼

Header: 좌측 위 검 색창, 중앙 로고, 우측 위 SNS(Hover 사용)등으로 구성됨

Main: 제품, E-Sports, 혁신, 뉴스, 지원 탭(Hover 사용)으로 구성된 메인화면으로 지원탭을 제외한 나머지 탭을 Click 시 해당 페이지로 이동

제품: 좌측 다섯가지 탭메뉴(마우스, 키보드, 사운드장치, 가구, 악세서리)로 구성되있으며 각 탭 Click 시 해당 제품 페이지로 이동함

중앙 사진을 통해 해당 모델의 외형을 확인할 수 있고 밑에 있는 버튼들을 통하여 해당 모델의 외형을 다양한 각도에서 확인할 수 있음

중앙 하단의 버튼을 통하여 다음 모델을 볼 수 있고 우측 상단에 있는 Color 버튼을 통하여 해당 모델의 다른 색상을 확인할 수 있으며

우측 하단의 버튼을 통해 해당 제품의 상세정보, 구성품 등을 볼 수 있음, 좌측 상단 X 버튼을 누르면 메인으로 이동됨

E-Sports: 후원하고 있는 E-Sports팀과 간단한 설명 표시되어있음, X 버튼 누를 시 메인으로 이동

혁신: 현재 로지텍 G가 가지고 있는 기술등을 표시하고 있으며 각 탭 Hover 시 해당 기술에 대한 간단한 설명이 나옴. X 버튼 누를 시 메인으로 이동

뉴스: 로지텍 G 관련 뉴스를 나열하고 있으며 Hover 시 해당 마우스가 어느 기사 위에 있는지 더 잘보이도록 하였음. X 버튼 누를 시 메인으로 이동

Footer: 하단의 화살표를 Click 시 높이 165px정도의 Footer가 나오며 다시 Click 시 원래 크기로 되돌아감.
